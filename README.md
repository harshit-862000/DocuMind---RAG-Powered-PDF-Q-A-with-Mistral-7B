# DocuMind---RAG-Powered-PDF-Q-A-with-Mistral-7B
A RAG app that lets users upload PDFs and ask questions in natural language. Powered by Mistral-7B-Instruct (8-bit quantized), HuggingFace sentence-transformers for embeddings, and LlamaIndex for vector indexing — all runnable on a free Google Colab GPU.
